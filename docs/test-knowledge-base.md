# Тестирование базы знаний

## ✅ Исправлено

1. База знаний скопирована в `frontend/public/knowledge-base/`
2. Сервис использует fetch для загрузки файлов
3. Добавлено логирование для отладки

## Как проверить

### 1. Запустите приложение
```bash
cd frontend
npm run dev
```

### 2. Откройте браузер
- Перейдите на страницу AI чата
- Откройте консоль (F12 → Console)

### 3. Проверьте логи
Должны увидеть:
```
✅ AI готов к работе
✅ База знаний загружена: ~14000 символов
📚 Первые 300 символов: # О колледже Narxoz...
```

### 4. Задайте тестовые вопросы

**Вопрос:** "Какой адрес колледжа?"
**Ожидаемый ответ:** г. Алматы, 10-й микрорайон, д. 7А

**Вопрос:** "Во сколько начинаются занятия?"
**Ожидаемый ответ:** 08:00

**Вопрос:** "Сколько стоит общежитие?"
**Ожидаемый ответ:** 25 000 тг/месяц

**Вопрос:** "Кто директор колледжа?"
**Ожидаемый ответ:** Абайдуллаев Мақсат Серікболұлы

### 5. Проверьте в консоли
При отправке сообщения должен быть лог:
```
📤 База знаний добавлена в контекст: ~14000 символов
```

## Если не работает

1. Проверьте, что файлы скопированы:
```bash
dir frontend\public\knowledge-base
```

2. Проверьте консоль на ошибки 404

3. Убедитесь, что Groq API ключ настроен в `.env`:
```
VITE_GROQ_API_KEY=your_key_here
```

## Обновление базы знаний

После изменения файлов в `knowledge-base/`:
```bash
robocopy knowledge-base frontend\public\knowledge-base /E
```

Или просто редактируйте файлы напрямую в `frontend/public/knowledge-base/`
