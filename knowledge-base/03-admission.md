# Поступление

## Документы для поступления
- Подлинник аттестата/диплома об образовании
- Медицинская справка формы 075-У (с флюорографией)
- Карта прививок по форме 063-У
- 6 фотографий 3х4
- Копия удостоверения личности и ИИН

## Раннее зачисление
Рекомендуется подавать документы в первой волне для гарантии места на популярных специальностях (Программирование, Право).

## Перевод из другого колледжа

### Шаги:
1. Получите **Транскрипт** в текущем колледже (список пройденных дисциплин)
2. Предоставьте его в Narxoz для расчета академической разницы
3. Получите «Письмо-запрос»
4. Заберите документы из старого колледжа
5. Завершите зачисление в Narxoz

### Важно:
Величина академической разницы влияет на семестр, с которого начнется обучение.
