# База знаний для Grok AI

## Как это работает

Grok автоматически использует информацию из папки `/knowledge-base/` при ответах на вопросы студентов.

## Структура (10 компактных файлов)

```
knowledge-base/
├── 01-about.md          # О колледже, контакты, цифры
├── 02-specialties.md    # Специальности и сроки обучения
├── 03-admission.md      # Поступление и перевод
├── 04-schedule.md       # Расписание занятий
├── 05-pricing.md        # Тарифы и оплата
├── 06-academic.md       # Учебный процесс
├── 07-infrastructure.md # Инфраструктура (библиотека, общежитие)
├── 08-documents.md      # Документы и справки
├── 09-international.md  # Международные возможности
└── 10-faq-short.md      # Быстрые ответы на частые вопросы
```

## Как добавить новую информацию

### 1. Создайте .md файл в папке knowledge-base

Например: `knowledge-base/rules.md`

```markdown
# Правила колледжа

## Дресс-код
- Деловой стиль одежды
- Запрещены шорты и майки

## Посещаемость
- Обязательное посещение всех занятий
- При пропуске нужна справка
```

### 2. Перезапустите приложение

```bash
npm run dev
```

База знаний загрузится автоматически!

## Преимущества новой структуры

✅ **Компактные файлы** - каждый файл 200-500 слов
✅ **Быстрая загрузка** - Grok обрабатывает информацию эффективнее
✅ **Легко обновлять** - изменения в одном файле не затрагивают другие
✅ **Четкая структура** - легко найти нужную информацию
✅ **Нет дублирования** - каждая тема в отдельном файле

## Что можно добавить

- Правила колледжа
- Информация о преподавателях
- Описание специальностей
- Инструкции по использованию систем
- Контакты отделов
- Расписание мероприятий
- Любую другую полезную информацию

## Технические детали

- Все .md файлы загружаются автоматически при старте
- Информация передается в Grok вместе с каждым запросом
- Grok использует эту информацию как контекст для ответов
- Не требуется обучение модели - работает сразу!

## Ограничения

- Каждый файл: 200-500 слов (оптимально для Grok)
- Общий размер базы: ~5000 слов
- Используйте четкую структуру с заголовками
- Избегайте дублирования информации

## Обновление информации

Просто отредактируйте нужный .md файл и перезапустите приложение:
```bash
npm run dev
```

Изменения применятся мгновенно!
